Utilizando el evento: onclick

Un evento es un suceso que ocurre cuando se presiona una tecla, se presiona un botón, se dispone la flecha del mouse en una región, etc. Desde JavaScript podemos capturar estos eventos y efectuar un proceso (cambiar la fuente, abrir otra página, mostrar un mensaje etc.)

Los elementos que se muestran resaltados a continuación ejecutan una función con js al hacer clic sobre ellos, un parrafo, un elemento de una tabla y un boton, al poner el puntero sobre ellos el cursor cambia para indicar que se puede interactuar con el elemento
