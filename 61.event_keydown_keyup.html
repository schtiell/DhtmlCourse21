DHTML: HTML Dinámico Eventos de teclado: keydown & keyup

El evento keydown se activa cuando se pulsa la tecla. El evento keyup se activa cuando una tecla que se encuentra presionada se deja de pulsar.

Ejercicio

Se confeccionará una página que solicite un nombre de usuario el cual no puede tener el espacio en blanco. En caso que el operador ingrese dicho caracter mostraremos un mensaje y no agregaremos dicho caracter.
