DHTML: HTML Dinámico Evento: onload( ) y unload( )

El evento onload se dispara cuando se ha cargado un objeto, este evento es utilizado con mayor frecuencia dentro del elemento body, para ejecutar un script una vez que una página web ha cargado completamente todo el contenido

Es decir luego que se a creado el árbol con todos los elementos de la página es cuando ocurre este evento.

El evento onunload se ejecuta cuando salimos del sitio y nos dirigimos por medio de un hipervínculo a otro o cerramos la ventana del navegador.
