DHTML: HTML Dinámico Propiedad: innerHTML

Lapropiedad innerHTML crea automáticamente todos los elementos HTML y nodos de texto que le indicamos cuando le asignamos el valor. Es una de las formas más utilizadas para efectuar el cambio masivo de nodos de elementos y datos.

Para entender su funcionamiento se implementa una página que solicite el ingreso de dos url y las muestre seguidamente al presionar un botón. Lo resolverá utilizando dos metodologías, es decir con y sin la propiedad innerHTML.

La propiedad innerHTML se encarga de analizar todas las etiquetas HTML que contiene nuestro string y procede a crear tanto los ELEMENT_NODES como los TEXT_NODES
