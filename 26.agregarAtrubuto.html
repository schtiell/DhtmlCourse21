DHTML: HTML Dinámico Agregando atributos a nodos: setAttibute( )

El metodo setAttibute() establece el valor de un atributo en el elemento indicado. Si el atributo ya existe, el valor es actualizado, en caso contrario, el nuevo atributo es añadido con el nombre y valor indicado.

        Element.setAttribute(name, value);
