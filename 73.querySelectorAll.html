Accediendo a un conjunto de elementos HTML a traves de querySelectorAll

De la misma manera que se puede acceder a un elemento html a traves de un selector css, podemos también acceder a un conjunto de elementos HTML mediante el método querySelectorAll.

El método querySelectorAll devuelve una lista con todos los nodos del DOM que cumplen la regla que le pasamos como parámetro.

Hay que tener en cuenta que querySelectorAll retorna un objeto de la clase NodeList.
