DHTML: HTML Dinámico: Evento de desenfoque: blur

El evento blur se dispara cuando pierde el foco el control. Se puede capturar el evento blur de un control de tipo text, textarea, button, checkbox, file, password, radio, reset, submit.

Ejercicio

Para probar su funcionamiento se dispondrá dos controles de tipo text con algún contenido. Fijaremos de color azul su fuente. Al tomar foco el control cambiará a color rojo y al perder el foco volve
