DHTML: HTML Dinámico: Remover atributos: removeAttribute( )

El método removeAttribute( ) elimina el atributo con el nombre especificado del elemento.

        removeAttribute(attrName)

Donde attrName es una cadena que especifica el nombre del atributo para eliminar del elemento. Si el atributo especificado no existe, removeAttribute() regresa sin generar un error.

Debe usar removeAttribute() en lugar de establecer el valor del atributo null directamente o usando setAttribute(). Muchos atributos no se comportarán como se espera si los establece en null.
