DHTML: HTML Dinámico: Eventos del mouse: mousedown( ) & mouseup( )

Estos eventos se producen cuando el usuario hace clic en un botón del mouse. El evento mousedown se activa cuando el botón de un dispositivo apuntador (usualmente el botón de un ratón) es presionado en un elemento.

MouseUp se produce cuando el usuario suelta el botón del mouse.

La propiedad target de la interfaz del event.currentTarget es una referencia al objeto en el cual se lanzo el evento.

Ejemplo:

Implementar una tabla con tres fila y tres columnas, al presionar el botón del mouse dentro de una casilla de la tabla cambiar el color de fondo y cuando levantamos el dedo del mouse regresar al color original de la casilla.

Tambien debe enviar a consola el id del elemento que disparó el evento y el valor del color por el que fue cambiado

Property Type Descrition
target EventTarget The event target (the topmost target in the DOM tree)
type DOMString The type of event
bubbles Boolean Whether the event normally bubbles or not
