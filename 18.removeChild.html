DHTML: HTML Dinámico: Eliminar y verificar nodos hijo: removeChild & hasChildNodes

Para eliminar un nodo de texto disponemos de un método llamada: removeChild

Para eliminar un nodo de texto primero debemos obtener la referencia de un elemento html

Otro método importante que contiene todo nodo de tipo elemento es: hasChildNodes, la misma retorna true en caso que el nodo tenga nodos hijos (sean nodos de tipo texto o nodos de tipo elemento).
