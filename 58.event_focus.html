El evento focus se produce cuando se activa el control. Se puede capturar el evento focus de un control de tipo text, textarea, button, checkbox, file, password, radio, reset, submit, search, href etc.

Su sintaxis:

        addEventListener('focus', (event) => { instrucctions... });
