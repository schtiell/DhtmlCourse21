DHTML: HTML Dinámico: Eventos del mouse: mouseover( ) y mouseout ( )

El evento mouseover se dispara cuando se posiciona la flecha del mouse a un elemento HTML y el mouseout cuando se saca la flecha del elemento

Ejemplo:

Ejercicio: Crear una tabla con tres filas y tres columnas cambiar el color del interior de la casilla cuando ingresamos con el mouse. Utilizando el metodo getElementsByTagName()

Property Type Descrition
target EventTarget The event target (the topmost target in the DOM tree)
type DOMString The type of event
bubbles Boolean Whether the event normally bubbles or not

Ejercicio: Para probar estos dos eventos utilizando la especificación del DOM W3C se implementará una página que contenga tres botones y cuando ingrese la flecha del mouse al botón cambiaremos el color de texto del mismo, retornando al color original cuando retiramos la flecha del control.
